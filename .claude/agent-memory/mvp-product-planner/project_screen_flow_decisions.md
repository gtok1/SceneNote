2026-05-02 기준 확정된 화면 흐름 결정사항.

## 네비게이션 구조

- 하단 탭바 3개: 홈/라이브러리 (탭1), 콘텐츠 검색 (탭2), My Page (탭3)
- Expo Router file-based routing 사용
- 핀 생성/편집은 SCR-010 (타임라인 핀 생성/편집)이 생성·편집 모드를 겸용

## 화면 수 (SCR-001 ~ SCR-014)

- MVP 포함: SCR-001, 002, 003, 004, 005, 006, 007, 008, 009, 010, 011, 013, 014 (13개)
- MVP 제외 (Phase 2): SCR-012 (명장면 모음/Highlight Reel)

## 핵심 UX 결정

| 결정 | 내용 |
|------|------|
| 라이브러리 추가 방식 | 바텀 시트(모달) 방식, 상태 선택 즉시 저장 |
| 핀 목록 UI | 세로 리스트형 (진행바형은 Phase 2) |
| 검색 결과 | SCR-004와 SCR-005는 동일 화면의 상태 전환으로 처리 |
| 에피소드 진행률 표시 | 시즌별 접이식 또는 탭 구조 (디자이너 결정 필요) |
| 영화 콘텐츠 | 에피소드 선택 화면(SCR-009) 진입 불가, SCR-008에서 직접 핀 생성 |

**Why:** 화면 수가 많아 일관된 구조 없이 설계하면 네비게이션 복잡도가 급증함.
**How to apply:** 새 화면을 추가할 때 탭바 구조와 이 결정사항을 먼저 참조.
