SceneNote 기획 문서에서 사용하는 표준 용어가 2026-05-02 확정되었다.

## 확정 용어

| 표준 용어 | 사용 금지 표현 | 비고 |
|-----------|---------------|------|
| 핀 (Pin) | 마커, 태그(단독), 메모 | 타임라인 핀의 단축 호칭 |
| 타임라인 핀 | 타임스탬프 메모, 핀포인트 | 공식 기능 이름 |
| timestamp_seconds | 타임코드, 시간 인덱스 | DB 컬럼명, 정수 타입 |
| My Library | 라이브러리, 내 목록 | 화면/기능 명칭 |
| 감상 상태 | 시청 상태, 뷰 상태 | status 필드 |
| 보고 싶음 | 위시리스트, 찜 | wishlist 상태 |
| 보는 중 | 시청 중, 진행 중 | watching 상태 |
| 완료 | 다 봄, 시청 완료 | completed 상태 |
| 드랍 | 포기, 중단 | dropped 상태 |
| 에피소드 진행률 | 진도, 시청 진행 | user_episode_progress |
| 스포일러 플래그 | 스포 태그 | is_spoiler 컬럼 |

## 감정(Emotion) 레이블 표준 (10종)
감동, 설렘, 긴장, 놀람, 즐거움, 슬픔, 분노, 공포, 공감, 최고

**Why:** 여러 문서 간 용어 불일치를 방지하고 개발팀과의 소통 명확성 유지.
**How to apply:** 새 문서 작성 또는 기존 문서 수정 시 이 표를 참조.
